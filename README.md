# 📊 Delhivery Small World Dashboard

A modern and interactive **Power BI dashboard** designed to analyze business performance across revenue, customers, profit, and returns.

---

## 🚀 Project Overview

This dashboard provides a **comprehensive business intelligence view** of sales and customer data. It helps stakeholders quickly understand:

- Revenue trends 📈  
- Customer behavior 👥  
- Profit insights 💰  
- Return analysis 🔄  

Built with a focus on **clarity, usability, and decision-making**.

---

## 🧩 Features

- 📅 Year Filter (2024 & 2025)
- 📊 Revenue Trend Analysis (Monthly)
- 🧮 Key Metrics Cards:
  - Total Revenue  
  - Total Profit  
  - Total Customers  
  - Return Rate  
- 🌍 Regional Performance Analysis
- 🛍️ Category-wise Revenue & Returns
- 🧾 Top Products & Customers Insights
- 📦 Sub-category Return Breakdown

- Return rate is relatively low (~5%) ✔️  
- Binders & Chairs have higher return counts
